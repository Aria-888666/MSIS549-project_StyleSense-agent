### Overview
StyleSense is an AI-powered styling assistant that generates outfit recommendations based on weather, occasion, and user preferences.

### Features
- Context-aware outfit generation
- Keyword constraint parsing (e.g., "no jeans")
- Three independent outfit options
- Virtual try-on
- Shopping suggestions

### Tech Stack
- Lovable (frontend)
- OpenAI API
- Lovable Cloud & Supabase Edge Functions
- Framer Motion
